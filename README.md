使用说明(Instructions):

1、请直接下载PCB中的Gerber文件。然后在嘉立创下单助手（链接在项目中已标注出）中下单PCB，上传Gerber文件。 First，Gerber!

2、一定要勾选“沉金”的选项，建议红色板，红金搭配才是这块板子最初设计的颜色，若有其他审美，可以自行决定   
Second, take any color what you want.But the "Golden" is the necsssity. Red and "Golden" is most suitable decision which is my original design.
